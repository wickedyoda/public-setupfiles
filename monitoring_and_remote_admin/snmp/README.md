# snmp

## Overview
SNMP setup scripts for Linux, macOS, and OpenWrt along with default configuration references.

## Contents
- `Setup_snmp_macos.sh`
- `openwrt_snmp_defaults.txt`
- `setup_snmp.sh`
- `setup_snmp_openwrt.sh`
- `setup_snmp_openwrt_publicv2.sh`
- `setup_snmp_openwrt_v2.sh`

## Usage
Use the script that matches the target platform and validate the SNMP version and community or credential model first.

## Notes
SNMP settings expose monitoring data, so align them with the security posture of the network.
