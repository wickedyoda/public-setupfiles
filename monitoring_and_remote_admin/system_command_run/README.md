# system_command_run

## Overview
Tools for running commands across multiple machines using shell and Python helpers with a shared machine list.

## Contents
- `Run_command_on_machines.sh`
- `machines.txt`
- `run_command_on_machines.py'`
- `run_command_on_machines_Keys.py'`

## Usage
Review the machine list and command runner settings before using this folder on a live environment.

## Notes
Remote command execution can be high impact, so start with safe commands and verify connectivity assumptions.
