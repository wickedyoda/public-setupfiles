# observium

## Overview
Observium installation helpers plus the agent-specific setup scripts stored in the `observium_agent` subfolder.

## Contents
- `custom_install.sh`
- `observium_agent`

## Usage
Use this folder when deploying Observium or its agent configuration on supported hosts.

## Notes
Observium setup often depends on web, database, and SNMP prerequisites that should be prepared in advance.
