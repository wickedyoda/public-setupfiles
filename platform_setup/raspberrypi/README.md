# raspberrypi

## Overview
Raspberry Pi automation scripts for mounts, package installs, file sharing, and data movement.

## Contents
- `Media_to_ap-share.sh`
- `public_to_usb.sh`
- `rasp_pi_install_apps.sh`
- `raspberry_pi_fstab.sh`

## Usage
Use these scripts on Raspberry Pi systems after checking device-specific paths and service expectations.

## Notes
Embedded or ARM systems can differ from standard Linux hosts, so validate each script on the target hardware.
