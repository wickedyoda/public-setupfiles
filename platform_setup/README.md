# platform_setup

## Overview
This category groups platform-specific installation and setup scripts for Linux distributions, Raspberry Pi, macOS, OpenWrt, Cockpit, and Paperless-ngx.

## Contents
- `Kali_tools_install`
- `cockpit`
- `debian-dist-upgrades`
- `debian-files`
- `kali-purple`
- `mac-scripts`
- `openwrt_scripts`
- `paperless-ngx`
- `raspberrypi`
- `server_config`
- `ubuntu-based`

## Usage
Use the subfolder that matches the target platform or service and review prerequisites before execution.

## Notes
Many scripts here install packages or change host configuration, so they should be tested on non-production systems first.
