# mac-scripts

## Overview
macOS utility scripts for Finder behavior, rsync tasks, package maintenance, and related automation.

## Contents
- `brewscripts`
- `disable_DS_store.sh`
- `not working- download-sync.sh`
- `rsync basic.sh`

## Usage
Use the script that matches the macOS task you want to automate.

## Notes
macOS path, permission, and launch behavior can differ from Linux, so test each workflow locally first.
