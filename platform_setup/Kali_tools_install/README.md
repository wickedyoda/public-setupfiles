# Kali_tools_install

## Overview
Scripts for installing Kali tooling on compatible Debian-based systems.

## Contents
- `Kalitoolsinstall.sh`
- `setup_debian_kali_tools.sh`

## Usage
Use these scripts when you want a targeted Kali toolset without fully converting the base operating system.

## Notes
Package availability and repository choices should be reviewed before execution.
