# copy-move-files

## Overview
Shell scripts for copying and moving data between public shares, NAS storage, local media, and USB targets.

## Contents
- `Brandons-Minecraft_backup.sh`
- `media_to_usb.sh`
- `nas public.sh`
- `public_to_usb.sh`

## Usage
Review source and destination paths before running any script in this folder.

## Notes
These scripts are designed around specific storage layouts and likely need path adjustments on new systems.
