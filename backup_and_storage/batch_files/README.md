# batch_files

## Overview
Windows batch and PowerShell scripts for backups, sync jobs, drive mapping, and file movement between local disks, NAS shares, and USB storage.

## Contents
- `ABC---Backup VM's.bat`
- `Backup Videos.bat`
- `Copy myvideos from nas to usb.bat`
- `Downloads Backup.bat`
- `Education to education nas.bat`
- `IT work .bat`
- `Map Drives.bat`
- `Murder VM Backup.bat`
- `NAS to 4tb Local USB.bat`
- `Older Files`
- `Pictures backup.bat`
- `Skyrim Copy from Nas to Local.bat`
- `Skyrim Copy to nas.bat`
- `Sync Private Folders.bat`
- `Traver-desk VM Backup.bat`
- `coding and programming backup.bat`
- `familymedicalbackup.bat`

## Usage
Use these scripts on Windows systems after adjusting drive letters, UNC paths, and local folder targets.

## Notes
The `Older Files` folder contains archived variants and older experiments kept for reference.
