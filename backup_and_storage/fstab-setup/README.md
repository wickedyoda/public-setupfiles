# fstab-setup

## Overview
Helpers and notes for building or updating Linux `fstab` mount entries.

## Contents
- `Fstab entries.txt`
- `setup-fstab-remake-mnt.sh`
- `setup-fstab.sh`

## Usage
Use these scripts and notes when preparing persistent mount configurations on Linux hosts.

## Notes
Incorrect `fstab` changes can block boot or mount behavior, so validate entries carefully.
