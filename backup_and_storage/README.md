# backup_and_storage

## Overview
This category groups backup jobs, file-move helpers, storage mount configuration, and desktop/document redirection utilities.

## Contents
- `Docker_backup`
- `batch_files`
- `copy-move-files`
- `copy_palworld_local`
- `fstab-setup`
- `redirecting-desktop_and_documents`

## Usage
Open the project folder that matches the workflow you want, then review the contained scripts before running them.

## Notes
Most scripts here move or copy user data. Validate source and destination paths before using them on a live system.
